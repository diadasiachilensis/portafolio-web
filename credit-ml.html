Proyecto 1: Análisis de Riesgo Crediticio con Regresión Lineal

Resumen

Este proyecto consistió en aplicar técnicas de análisis exploratorio de datos y modelado predictivo al dataset German Credit. El objetivo principal fue identificar variables relevantes en la evaluación de solvencia crediticia y construir un modelo de regresión lineal para predecir el riesgo asociado a los solicitantes de crédito.

Contexto / Problema

El dataset German Credit contiene información financiera y demográfica de solicitantes de crédito. El desafío fue analizar y modelar estos datos para comprender qué factores explican mejor la probabilidad de riesgo crediticio. Esta problemática es central en el sector bancario, ya que influye en la toma de decisiones de otorgamiento de préstamos.

Enfoque

  • Limpieza de datos: tratamiento de valores nulos, codificación de variables categóricas.
  • Análisis exploratorio: visualización de distribuciones, identificación de outliers y correlaciones.
  • Modelado: entrenamiento de un modelo de Regresión Lineal usando Scikit-learn.
  • Evaluación: métricas de desempeño Mean Squared Error (MSE) y R².

Reflexión / Aprendizajes

Este proyecto reforzó mis habilidades en preprocesamiento y exploración visual de datos. Una dificultad importante fue el tratamiento de datos faltantes y outliers, que afectaban la calidad del modelo. Como mejora futura, propondría aplicar modelos más robustos (ej. árboles de decisión, Random Forest, Gradient Boosting) y técnicas de validación cruzada para mejorar la capacidad predictiva.
